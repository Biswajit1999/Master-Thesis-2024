# Active Optics Calibration Report (2024-2026)

This directory contains the editable LaTeX source and compiled PDF for the combined EXOhSPEC active-optics calibration report.

## Main outputs

- `AO_Combined_Final_Report_2024_2026.pdf` - compiled technical report
- `latex/main.tex` - main LaTeX source
- `latex/parts/` - editable report sections

The GitHub Actions workflow compiles the LaTeX source with pdfLaTeX and commits the resulting PDF automatically whenever the report source changes.
